## üöÄ Getting Started

### Prerequisites
- Node.js 18+
- pnpm
- Sui CLI

### Installation
```bash
# Install dependencies
pnpm install

# Start development servers
pnpm run dev
```

### Smart Contract Deployment
```bash
# Deploy CheckIn contract
cd contracts/CheckIn
sui move build
sui client publish

# Deploy Passport contract
cd ../passport-epfl
sui move build
sui client publish

# Deploy GrantPool contract
cd ../lemanflowContracts
sui move build
sui client publish
```
